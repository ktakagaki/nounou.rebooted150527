RandomAccessFileBE

class RandomAccessFileBE extends DataInput with Closeable

Wrapper for java.io.RandomAccessFile.

The main differences to java.io.RandomAccessFile are in the readXXX(n: Int) functions, which will try to read n samples from the file. Try to use these function instead of reading multiple times in a loop with readXXX(), as each read is costly in terms of performance.

Of note, RandomAccessFileBE explicitly supports big endian reading/writing (as does java.io.RandomAccessFile). Use nounous.io.RandomAccessFileLE to read/write little endian.

Each function throws a java.io.IOException, which can be caught in Scala if necessary to detect ends of files, for example. Catching is obligatory in Java.
