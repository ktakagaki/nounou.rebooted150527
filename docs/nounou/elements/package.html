    Database to encapsulate marked events in data recording.

    Database to encapsulate marked events in data recording. Events are stored as NNEvent objects, which encapsulate timestamp, duration, code, and comment string. An NNEvents database consists of TreeSet(s) of XEvents. Each event "port" has its own TreeSet---this allows different ports to have events with exactly the same timing. (TreeSet's with black-red trees cannot support keys with equivalent sorting values)
