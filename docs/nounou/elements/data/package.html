  1. abstract class NNData extends NNElement with NNChannelsElement with NNDataScaleElement with NNDataTimingElement

    Base class for data encoded as Int arrays, this is the main data element for an experiment, whether it be electrophysiolgical or high-sampling-rate imaging.

    Base class for data encoded as Int arrays, this is the main data element for an experiment, whether it be electrophysiolgical or high-sampling-rate imaging.

    This object is mutable, to allow inheritance by nounou.elements.data.filters.NNDataFilter. For that class, output results may change, depending upon _parent changes. Each trace of data must share the following variables: sampling, start, length, xBits, absGain, absOffset, absUnit

    This trait is a simple marker Trait for NNData, which indicates that the data is auxilliary. This means, for instance for imaging data, parallel continuous data such as stimulus signals, which are simulatneously digitized.
